# Telegram Trading Bot

AI stock analysis bot for Telegram. Deploys to Render/Railway/Heroku.

## Deploy to Cloud

1. Upload to GitHub
2. Connect repo to Render/Railway
3. Add environment variable: `BOT_TOKEN=your_telegram_token`
4. Deploy!

## Local Test

```bash
pip install -r requirements.txt
export BOT_TOKEN=your_token  # Linux/Mac
set BOT_TOKEN=your_token     # Windows
python main.py
```
